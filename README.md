# hydra_cluster

A k8s local cluster for laptops and desktops.

### For Ubuntu


```sh
$ git clone https://github.com/sanchil/hydra_cluster.git
$ cd hydra_cluster
$ git checkout ubuntu
```


### For Centos [Work in Progress]

Please wait up for a while for centos.

```sh
$ git clone https://github.com/sanchil/hydra_cluster.git
$ cd hydra_cluster
$ git checkout centos
```
